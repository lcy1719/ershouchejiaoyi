## 【功能介绍】
**本系统包含管理员、商家、用户三个角色。
管理员角色：商家管理、用户管理、汽车管理、汽车留言管理、汽车收藏管理、公告信息管理、论坛管理、修改密码、个人中心、公告类型管理、汽车品牌管理、基础数据管理。
商家角色：汽车管理、汽车留言管理、汽车收藏管理、修改密码、个人中心、公告信息查看、论坛管理。
用户角色：汽车信息查询、汽车收藏、汽车留言、更改个人信息、个人中心、在线论坛、公告信息查看、评论帖子、发布帖子。**
